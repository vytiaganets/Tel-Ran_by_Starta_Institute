1) Обдумайте проект, который бы Вы хотели написать как самостоятельный дипломный проект с использование Spring Boot. Сделайте его краткое описание.
Создайте шаблон приложения с использованием сайта https://start.spring.io/. Подумайте какие Dependencies можно было бы сразу включить в Ваш шаблон.
В корневом каталоге проекта (там же, где находиться файл pom.xml) создайте новый файл README.md и вставьте в него описание Вашего проекта.
